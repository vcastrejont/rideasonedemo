### Installing the app

#### Requirments:
 - Nodejs
 - MongoDB

*Installing Back end components:*
```sh
$ npm  install
```
 
*Installing Front end components:*
```sh
$ bower install
```
#### Run carpooling app

*execute the next command to run the app*
```sh
$ npm start
```

*for front end changes run the watcher *
```sh
$ npm  run watch
```

*Open your browser at:*
```sh
http://localhost:3000/
```
